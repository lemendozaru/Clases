1.2 Uso correcto de la calculadora para expresiones aritméticas

Cómo evitar errores al introducir operaciones

Error común #1: Olvidar paréntesis en el denominador

Expresión: \[ \frac{8 + 4}{3 - 1} \]

Forma incorrecta en calculadora básica:

8 + 4 ÷ 3 - 1 → La calculadora hace \(4 \div 3\) primero.

Forma correcta:

(8 + 4) ÷ (3 - 1) = \(12 \div 2 = 6\)

Error común #2: Signos negativos y exponentes

Expresión:

\[ (-3)^2 \quad \text{vs} \quad -3^2 \]

En calculadora:

  • (-3)^2 = 9
  • -3^2 = -9 (primero eleva, luego aplica el signo)

Error común #3: Falta de paréntesis en raíces o fracciones largas

Expresión:

\[ \frac{5 + 7}{2 \times 3} \]

Correcto: (5 + 7) ÷ (2 × 3) = \(12 \div 6 = 2\)

Ejemplo práctico con calculadora científica

Expresión:

\[ \frac{10 + 2 \times 3}{4 - (6 \div 2)} + (-2)^3 \]

Secuencia de tecleo (modo lineal o natural):

  1. Numerador: 10 + 2 × 3 = 16
  2. Denominador: 4 - (6 ÷ 2) = 4 - 3 = 1
  3. Fracción: 16 ÷ 1 = 16
  4. Exponente: (-2)^3 = -8
  5. Suma final: 16 + (-8) = 8

Respuesta: 8

Secuencia directa ((10 + 2 × 3) ÷ (4 - (6 ÷ 2))) + (-2)^3

2.1 Ecuaciones lineales

Concepto clave

Una ecuación lineal tiene la forma:

\[ ax + b = c \]

Despejar significa aislar la variable usando operaciones inversas.

Ejemplo aplicado: Regresión lineal simple

Modelo predictivo:

\[ y = \beta_0 + \beta_1 x \]

Si la recta debe pasar por el punto de medias \((\bar{x}, \bar{y})\):

\[ \bar{y} = \beta_0 + \beta_1 \bar{x} \]

Despejamos \(\beta_0\):

\[ \beta_0 = \bar{y} - \beta_1 \bar{x} \]

Ejercicio 1 (resuelto)

Dados \(\bar{x} = 4\), \(\bar{y} = 10\) y \(\beta_1 = 1.5\), encuentra \(\beta_0\).

Solución:

\[ \beta_0 = 10 - (1.5 \times 4) = 10 - 6 = 4 \]

Ejercicio 2 (propuesto)

Despeja \(r\) en la fórmula del interés compuesto:

\[ A = P(1 + r)^t \]

Ejercicio 3 (resuelto - con fracciones)

Resuelve para (x):

\[ \frac{2x - 1}{3} = 5 \]

Solución:

  1. Multiplicar por 3: \[ 2x - 1 = 15 \]
  2. Sumar 1: \[ 2x = 16 \]
  3. Dividir por 2: \[ x = 8 \]

2.2 Sistemas de ecuaciones lineales (2x2)

Aplicación directa: Balanceo de recursos o recomendaciones

Ejemplo: Un sistema de recomendación usa dos factores \(a\) y \(b\):

\[ 2a + 3b = 4.5 \]

\[ a - b = 0.5 \]

Método de reducción (resuelto paso a paso):

  1. Multiplicamos la segunda ecuación por 3: \[ 3a - 3b = 1.5 \]

  2. Sumamos con la primera ecuación: \[ (2a + 3b) + (3a - 3b) = 4.5 + 1.5 \]

    \[ 5a = 6 \implies a = 1.2 \]

  3. Sustituimos \(a\) en \(a - b = 0.5\): \[ 1.2 - b = 0.5 \implies b = 0.7 \]

Solución: \(a = 1.2\), \(\ b = 0.7\)

Ejercicio 5 (resuelto)

Resuelve el siguiente sistema usando sustitución:

\[ x + 2y = 5 \] \[ 3x - y = 1 \]

Solución:

  1. De la primera: \(x = 5 - 2y\)
  2. Sustituir en segunda: \(3(5 - 2y) - y = 1\)
  3. \(15 - 6y - y = 1 \implies 15 - 7y = 1\)
  4. \(-7y = -14 \implies y = 2\)
  5. \(x = 5 - 2(2) = 1\)

Respuesta: \(x = 1, y = 2\)

2.3 Ecuaciones cuadráticas

Forma general:

\[ ax^2 + bx + c = 0 \]

Vértice (mínimo o máximo):

\[ x_v = -\frac{b}{2a} \]

Fórmula general (raíces):

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

Ejemplo aplicado: Función de costo

\[ C(x) = x^2 - 6x + 9 \]

Encontrar el mínimo:

\[ x_v = -\frac{-6}{2 \times 1} = \frac{6}{2} = 3 \]

Valor mínimo:

\[ C(3) = 9 - 18 + 9 = 0 \]

Ejercicio 8 (resuelto)

Encuentra el mínimo de:

\[ C(x) = 2x^2 - 8x + 5 \]

Solución:

\[ x_v = -\frac{-8}{2 \times 2} = \frac{8}{4} = 2 \]

\[ C(2) = 2(4) - 16 + 5 = 8 - 16 + 5 = -3 \]

2.4 Desigualdades lineales

Ejemplo: Región factible

Un problema de recursos tiene:

\[ 3x + 2y \leq 12 \]

\[ x \geq 0,\ y \geq 0 \]

Interpretación geométrica:

  • Si \(y = 0 \rightarrow 3x \leq 12 \rightarrow x \leq 4\)
  • Si \(x = 0 \rightarrow 2y \leq 12 \rightarrow y \leq 6\)

La región factible es un triángulo con vértices en \((0,0)\), \((4,0)\) y \((0,6)\).
